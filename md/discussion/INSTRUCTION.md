## 🎯 Назначение

Эта папка — **рабочее пространство для общения двух AI агентов** (Qwen и Codex) по задачам проекта. Человек читает, утверждает, принимает решения.

## 📝 Формат файла задачи

### Шапка (метаданные)

```markdown
# Discussion #N: Название задачи

| Метаданные | Значение |
|------------|----------|
| **Статус** | 🟡 Open |
| **Приоритет** | 🔴 Высокий |
| **Создано** | 2026-03-30 |
| **Постановщик** | 🤖 AI Qwen |
| **Исполнитель** | 🤖 AI Codex |
| **Оценка** | 4-8 часов |
```

---

### План задачи (обязательно)

```markdown
## 📋 План задачи

### Этап 1: Название этапа

**Цель:** [что нужно достичь]

**Команды для выполнения:**
```bash
[команды для терминала]
```

**Ожидаемый результат:**
- [ ] Чекбокс 1
- [ ] Чекбокс 2
```

---

### Обсуждение (обязательно)

```markdown
## 💬 Обсуждение задачи

### 🤖 AI Qwen (Постановщик) — 2026-03-30

**Рассуждения по постановке:**

[текст рассуждений]

**Вопрос к AI Codex:**

[вопрос]

---

### 🤖 AI Codex (Исполнитель) — 2026-03-30

**Рассуждения по реализации:**

[текст анализа]

**Вопросы к AI Qwen:**

- [вопрос 1]
- [вопрос 2]

---

### 👤 Человек — 2026-03-30

**Решения:**

- Выбранный вариант: [A / B / C / D]
- Комментарий: [текст]
```

## 🚀 Быстрый старт

### Для AI Qwen (создание новой задачи)

```
1. Скопируй discussion-1-*.md как шаблон
2. Измени название и метаданные
3. Опиши план по этапам
4. Напиши свои рассуждения в обсуждении
5. Задай вопросы к AI Codex
```

### Для AI Codex (начало работы)

```
1. Открой файл задачи
2. Прочитай план и рассуждения AI Qwen
3. В секции обсуждения напиши:
   - Анализ плана
   - Вопросы
   - Предложения
4. Жди утверждения от человека
```

### Для Человека (утверждение)

```
1. Прочитай диалог AI в секции обсуждения
2. В своём блоке напиши:
   - Выбранный вариант стратегии
   - Решение по вопросам AI
3. Отметь галочкой ✅ план
```

## 📊 Статусы задач

| Статус | Иконка | Значение |
|--------|--------|----------|
| Open | 🟡 | В обсуждении / реализации |
| Done | ✅ | Завершена, принята человеком |
| Blocked | ⛔ | Ждёт решения (указать причину) |
| Cancelled | ❌ | Отменена |

## 🎯 Правила хорошего тона

### Для AI

| Правило | Описание |
|---------|----------|
| **Пиши рассуждения** | Не только "что делать", но и "почему так" |
| **Задавай конкретные вопросы** | Избегай открытых вопросов, предлагай варианты |
| **Фиксируй решения** | Если что-то изменилось — обнови план |
| **Уважай роль** | Не пиши за другую роль |

### Для Человека

| Правило | Описание |
|---------|----------|
| **Читай обсуждение** | AI уже обсудили нюансы, не дублируй |
| **Утверждай явно** | Пиши ✅ когда план одобрен |
| **Задавай контекст** | Если меняешь приоритеты — объясни почему |

## 🔧 Технические детали

### Как меня уведомить

| Команда | Что делаю |
|---------|-----------|
| `прочитай discussion-N` | Читаю файл задачи №N |
| `обнови задачу` | Читаю и предлагаю следующие шаги |
| `есть вопрос по задаче` | Читаю и отвечаю на вопрос |
| `завершаю задачу` | Помогаю с итоговым статусом |

## ❓ FAQ

**Q: Что если AI не согласны друг с другом?**  
A: Это нормально. AI Qwen и AI Codex могут предлагать разные варианты. Человек принимает финальное решение.

**Q: Можно ли пропустить секцию обсуждения?**  
A: Нет. Рассуждения помогают понять контекст и принять взвешенное решение.

**Q: Как часто обновлять статус задачи?**  
A: После каждого значимого этапа (диагностика → выбор стратегии → реализация → тесты → деплой).

**Q: Что делать если задача зависла?**  
A: Человек должен явно написать статус ⛔ Blocked с причиной в обсуждении.
