```

Мы закрыли это через Telegram mini-app: каталог + заявка в одном сценарии.
На запуск базовой витрины уходит около 10 минут.
Если нужно, скину рабочую ссылку на mini-app/бот с этим сценарием.
```

```
Делали такой же сценарий для продаж через Telegram:
1) подключили бота,
2) добавили товары/услуги,
3) запустили приём заявок.
Первый рабочий результат получили в тот же день.
Если интересно, отправлю ссылку на рабочий пример.
```

```
каталог -> заявка -> контакт с менеджером.
Главное, чтобы вход был в 1 клик и без лишних экранов.
```

## Reply Template (when someone asks for link)
```
Да, вот ссылка:
<YOUR_TRACKED_LINK>

Если хотите, после теста напишите, где именно упёрлись — подскажу по шагам.
```

## Anti-spam Rules
1. No mass copy-paste in the same wording.
2. Change first line every 2-3 comments.
3. No aggressive claims or fake numbers.
4. One CTA per message.
5. First provide value, then link.
