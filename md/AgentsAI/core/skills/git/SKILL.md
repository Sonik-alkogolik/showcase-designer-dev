# Git Skill

## Checklist

1. Проверить статус
2. Проверить diff
3. Добавить в stage только целевые файлы
4. Сделать осмысленный commit
5. Push по запросу

## Guardrails

- Не использовать destructive-команды без явного запроса
- Не переписывать историю без явного запроса
