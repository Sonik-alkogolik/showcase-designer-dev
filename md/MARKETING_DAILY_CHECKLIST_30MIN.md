# Daily Checklist (30 Minutes)

Updated: 2026-05-09

## Before Start (1 minute)
- Open:
  - `MARKETING_7D_SPRINT_INFOBIZ.md`
  - `MARKETING_COMMENT_TEMPLATES.md`
  - your weekly tracker CSV.

## Block 1: Find Discussions (5 minutes)
1. Pick 2-3 active threads.
2. Check:
- fresh messages today;
- real pain topic;
- allowed links format.

## Block 2: Publish Comments (20 minutes)
1. Post 4-6 comments total.
2. Use templates:
- `Template 1` and `Template 2` as base;
- edit first line each time.
3. Insert one tracked link per allowed thread.

## Block 3: Log Results (5 minutes)
1. For each posted comment, fill one row in tracker:
- source;
- link tag;
- replies;
- clicks;
- registrations.
2. Mark top-2 comments by signal:
- replies;
- clicks;
- registrations.

## End-of-Day Decision
1. Keep sources with signal.
2. Drop weak sources next day.
3. If day is even (`2, 4, 6`) change one variable:
- offer headline or opening sentence.
