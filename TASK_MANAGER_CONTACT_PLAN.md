# Task Plan: Manager Contact Flow (Without Online Payments)

Обновлено: 2026-03-29

## Цель

Сохранить существующую логику создания заказа, но заменить пользовательский сценарий оплаты на контакт с менеджером в Telegram.

## Task List

1. [x] Добавить в публичный API магазина данные контакта менеджера:
   - `manager_telegram_username`
   - `manager_telegram_url`
2. [x] Перевести checkout в WebApp на создание заказа без онлайн-оплаты:
   - всегда отправлять `create_payment: false`
3. [x] Добавить post-checkout CTA:
   - кнопка `Написать менеджеру`
   - ссылка вида `https://t.me/<username>?text=...`
4. [x] Обновить подписи в настройках магазина:
   - `notification_username` как контакт менеджера для клиента
5. [x] Обновить брендинг:
   - заменить `Showcase Designer` на `t-go` в UI

## Что осталось проверить вручную

1. В настройках магазина указан `notification_username` (например, `@my_manager`).
2. В WebApp оформить заказ:
   - заказ создаётся успешно;
   - показывается кнопка `Написать менеджеру`;
   - открывается чат Telegram с предзаполненным текстом.
3. Сценарий без `notification_username`:
   - заказ создаётся;
   - кнопка контакта менеджера не показывается.
