# Итоговая фиксация результата (auth + MoonShine)

Обновлено: 2026-04-18  
Файл правок: `Edits_18.04.2026_м_13.md`

## Что зафиксировано

1. MoonShine (подписки):
- в режиме редактирования подписки поле `Пользователь` больше не выпадающий список;
- в `edit` поле read-only;
- в `create` выбор пользователя через select сохранен.

2. Восстановление пароля:
- добавлен экран `Забыли пароль?` во фронте;
- добавлен email reset flow;
- добавлен Telegram fallback с одноразовым токеном и временем жизни;
- добавлен forced change password после входа по временному паролю.

3. Безопасность:
- введен флаг `must_change_password` для пользователя;
- добавлен middleware, который ограничивает доступ к API до смены временного пароля;
- после успешной смены флаг снимается.

## Статус

Блок задач завершен и зафиксирован в `Edits`.
