# Telegram Mini App: fullscreen просмотр изображения товара с оверлеем

Обновлено: 2026-04-23  
Файл правок: `Edits_23.04.2026_м_02.md`

## Что изменено

1. В mini-app добавлен просмотр изображения товара в полном размере.
2. Клик по изображению в карточке товара открывает fullscreen-режим.
3. Добавлен затемнённый фоновый оверлей.
4. Закрытие доступно:
- по кнопке `✕`,
- по клику на фон.

## Где работает

1. Список товаров (`catalog`).
2. Список избранного (`favorites`).
