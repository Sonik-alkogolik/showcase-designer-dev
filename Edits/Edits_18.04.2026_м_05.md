# Этап 4: UX для async-импорта (прогресс и нейтральные статусы)

Обновлено: 2026-04-18
Файл правок: `Edits_18.04.2026_м_05.md`

## Что улучшено

1. Убрана излишняя техническая формулировка в сообщениях для пользователя:
- вместо акцента на “очереди” показаны нейтральные статусы процесса импорта.

2. Добавлен визуальный индикатор загрузки на шаге результата:
- анимированная индикаторная полоса (indeterminate progress bar);
- показывается, пока импорт в статусе `pending` или `processing`;
- автоматически скрывается при `completed` / `failed`.

3. Улучшены состояния карточки результата:
- отдельный визуальный стиль для “в процессе”;
- иконка состояния: `⏳` во время обработки, `✅` после успеха, `❌` при ошибке.

4. Удалены лишние debug-логи из предпросмотра импорта.
